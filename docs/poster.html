Introduction & Abstract

The rapid advancement of digital technology creates a divide between urban and rural organizations. KEBANA currently relies on manual, paper-based methods and decentralized spreadsheets.

This project develops a Web-Based Management System exclusively for the KEBANA committee to establish a Single Source of Truth (SSoT) for membership, event archiving, and automated financial tracking.

Aligned with the Sarawak PCDS 2030, this system bridges the developmental gap, ensuring rural organizations participate fully in the digital economy.

Problem Statement

  • Geographical Dispersion: Committee members are scattered across Malaysia, making coordination difficult.
  • Lack of Centralization: Reliance on scattered Excel sheets and WhatsApp leads to data duplication, loss of institutional memory, and security risks.
  • Administrative Inefficiency: Manual calculations for finances and physical paper tracking consume excessive time and resources.

Objectives

  1. Centralized Database: Create a searchable, secure membership registry replacing manual records.
  2. Event Management: Develop a digital repository to create proposals and archive official documents.
  3. Financial Tracking: Implement a module for automated income/expenditure tracking and monthly report generation.

System Architecture & Core Modules

The KDMS is an internal backend tool focusing on operational efficiency, high-speed data entry, and role-based security.

Membership Registry

Comprehensive CRUD operations. Prevents duplicate ICs and serves as the Single Source of Truth.

Event Archiving

Draft proposals, upload approval letters, and store meeting minutes safely in the cloud.

Financial Ledger

A "Digital Cashbook" for the Treasurer. Auto-calculates balances and generates PDF statements.

Role-Based Access

Strict access tiers: Super Admin (President), Secretary (Members/Events), Treasurer (Finances).

Committee User
Modular PHP Core
Tailwind CSS / MYDS UI
MySQL Database
Cloud PaaS

MYDS-Inspired Design System

Ensures accessibility and standardized layout for committee members using Tailwind CSS, following the Malaysia Design System (MYDS) guidelines.

Methodology

Developed using the Agile Software Development Methodology to accommodate evolving requirements and digital literacy levels.

  • Iterative Feedback Loop: Sprints allow early demonstrations of modules.
  • Technology Stack: PHP for backend logic and session security, MySQL for relational database management, and Tailwind CSS (MYDS-inspired) for clean, responsive, and modern user interfaces.
  • Testing: Unit testing for financial ledger logic, UAT for usability validation.

Expected Outcomes

  • Reduction of administrative workload and search time from minutes to seconds.
  • Preservation of institutional memory against hardware damage or member resignation.
  • Enhanced transparency with auditable, auto-generated financial reports.

Significance & Conclusion

The KEBANA Digital Management System proves that Digital Transformation is accessible to rural NGOs. By utilizing a Low-Cost Sustainability approach (entry-level PaaS hosting), the system modernizes operations without burdening community funds.

It acts as a practical model for digital inclusivity, bringing Sarawak's rural communities into the data-driven era securely and efficiently.
